วางไฟล์ไอคอนของปลั๊กอินไว้ในโฟลเดอร์นี้
--------------------------------------
- ชื่อที่แนะนำ: icon.png (หรือ icon.svg)
- ปลั๊กอินจะมองหาไฟล์ตามลำดับ: resources/icon.png, resources/icon.svg,
  icon.png, icon.svg และถ้าไม่พบจะหยิบไฟล์ภาพไฟล์แรกในโฟลเดอร์ resources/ มาใช้แทน
- ค่า icon ใน metadata.txt ตั้งไว้ที่ resources/icon.png

(ผู้ใช้แจ้งว่าเตรียมไฟล์ไอคอนไว้แล้ว — เพียงนำมาวางไว้ที่นี่)
